    Filters¶

    scrapper.core.filters.cnpj_search_filter.CNPJSearchFilter ¶

    Bases: BaseFilter

    Esquema para o filtro de CNPJ.

    emitente_nfe: bool | None = False class-attribute instance-attribute ¶

    Se o CNPJ é emitente de NFE.

    grupo_objeto: str | None = None class-attribute instance-attribute ¶

    Código do grupo de objeto. Mais de um grupo pode ser passado, separados por vírgula. Ex: "1,2,3" para Obras, Serviços e Bens Patrimoniais. Veja a enumeração GrupoObjeto

    municipio: str | None = None class-attribute instance-attribute ¶

    Geocode do município da empresa, ex: "3304557" para o município do Rio de Janeiro. Veja: https://www.ibge.gov.br/explica/codigos-dos-municipios.php para mais detalhes.

    sancao_vigente: bool | None = False class-attribute instance-attribute ¶

    Se o CNPJ está sob sanção vigente.

    tipo_natureza_juridica: NaturezaJuridica = NaturezaJuridica.TODOS class-attribute instance-attribute ¶

    Tipo de natureza jurídica da empresa.

    uf_pessoa_juridica: str | None = None class-attribute instance-attribute ¶

    UF da pessoa jurídica da empresa no formato "XX". Ex: "SP", "RJ", "MG".

    valor_gastos_diretos_ate: float | None = None class-attribute instance-attribute ¶

    Valor máximo dos gastos diretos. Nota: Existe um bug no portal em que esse filtro não irá funcionar, mas a busca irá redirecionar para uma página de erro 400(?). Chamando a API diretamente, o filtro funciona.

    valor_gastos_diretos_de: float | None = None class-attribute instance-attribute ¶

    Valor mínimo dos gastos diretos. Nota: Existe um bug no portal em que esse filtro não irá funcionar, mas a busca irá redirecionar para uma página de erro 400(?). Chamando a API diretamente, o filtro funciona.

    valor_transferencia_ate: float | None = None class-attribute instance-attribute ¶

    Valor máximo da transferência. Nota: Existe um bug no portal em que esse filtro não irá funcionar, mas a busca irá redirecionar para uma página de erro 400(?). Chamando a API diretamente, o filtro funciona.

    valor_transferencia_de: float | None = None class-attribute instance-attribute ¶

    Valor mínimo da transferência. Nota: Existe um bug no portal em que esse filtro não irá funcionar, mas a busca irá redirecionar para uma página de erro 400(?). Chamando a API diretamente, o filtro funciona.

    scrapper.core.filters.cnpj_search_filter.NaturezaJuridica ¶

    Bases: str, Enum

    Enumeração para os tipos de natureza jurídica.

    Os valores são os códigos utilizados na API do Portal da Transparência.

    • 0: Todos
    • 1: Administração Pública
    • 2: Entidades Empresariais
    • 3: Entidades Sem Fins Lucrativos
    • 4: Organização Internacional

    scrapper.core.filters.cnpj_search_filter.GrupoObjeto ¶

    Bases: str, Enum

    Enumeração para os grupos de objeto.

    Os valores são os códigos utilizados na API do Portal da Transparência. - 1: Obras - 2: Serviços - 3: Bens Patrimoniais - 4: Materiais - 99: Outros
